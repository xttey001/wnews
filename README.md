# 悟空财经分析 (wnews)

基于五维进化交叉分析模型的财经新闻网站。

## 架构

### 五维进化模型

1. **悟空** - 深度新闻分析（S/A级优质新闻）
2. **沙僧** - 散户行为预测（全量新闻关键词分析，7天累积+时间衰减）
3. **白龙马** - 主力行为分析（K线+量价+悟空/八戒交叉验证）
4. **八戒** - 贝叶斯融合决策（融合沙僧+白龙马信号，输出胜率+ETF组合）
5. **唐僧** - 最终仲裁（跨层矛盾仲裁+全局风控，输出仓位建议）

### 三层交叉分析

```
原始新闻
   ↓
悟空（深度分析）
   ↓
沙僧（散户情绪）+ 白龙马（主力行为，含悟空/八戒信号）
   ↓
八戒融合（最终决策：融合四层信号）
   ↓
唐僧仲裁（跨层矛盾仲裁+全局风控）
```

- 白龙马 ← 悟空信号 + 八戒胜率（可信度调整）
- 八戒最终 ← 沙僧情绪 + 白龙马主力状态（胜率+行动调整）
- 唐僧 ← 跨层矛盾检测 + 全局风控（仓位系数调整）

## 数据格式

每日新闻 JSON 格式：
- `s_level`: S级新闻（主线级，影响1-4周）
- `a_level`: A级新闻（轮动级，影响1-3天）
- `douyin`: 抖音/平台相关
- `all_news`: 全量新闻（沙僧用）
- `wukong_judgment`: 悟空分析
- `wukong_enhanced`: 悟空增强版（v4进化系统）
- `bajie_conclusion`: 八戒融合结论（含沙僧/白龙马信号）
- `sangsha_module`: 沙僧散户情绪
- `white_dragon`: 白龙马主力行为
- `tang_sanzang`: 唐僧仲裁决策
- `evolution_v4`: 五维进化元数据

## 页面

- **财经首页**：`index.html` - 每日新闻+五维分析+决策矩阵
- **复盘中心**：`review.html` - 周度/月度复盘+进化知识库

## 技术栈

- 前端：纯 HTML + CSS + JavaScript（无框架）
- 后端：Python（位于 `news-evolution/` 目录）
- 部署：GitHub Pages + GitHub Actions
- 推送：网站 + 微信 + 飞书（三路推送）
